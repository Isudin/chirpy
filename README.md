# Documentation
## POST /api/users
### Creates a user

Request body:
```json
{
    "email": "test@example.com",
    "password": "Password"
}
```
Response body:
```json
{
    "id": "a4778791-5f83-4a3a-a9f8-6b0e88d9079a",
    "created_at": "2012-04-23T18:25:43.511Z",
    "updated_at": "2012-04-23T18:25:43.511Z",
    "email": "test@example.com",
}
```
